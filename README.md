# Cashio 💰

Cashio is a Personal Finance Tracker Web Application developed using Java, JSP, Servlets, JDBC, and MySQL.

## Features
- User Registration & Login
- Add Income
- Add Expenses
- Dashboard Analytics
- Financial Reports
- Income & Expense History
- Notifications
- Secure Session Management

## Technologies Used
- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML, CSS, JavaScript
